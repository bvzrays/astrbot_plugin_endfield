# 终末地协议终端 (astrbot_plugin_endfield) 更新日志

## 2.8.0 (2026-03-22)
- 🐛 **抽卡分析渲染**：修复无六星情况下的抽卡分析报错问题，以及未开启据点情况下的据点渲染报错问题
  
## 2.7.0 (2026-03-22)
- 🐛 **公告渲染**：修复了对于超长图文公告（如《塔卫二畅游指南》系列）由于渲染缩放和 Chrome 高度限制导致的渲染中途截断以及图片被挤压成细长条的问题。

## 2.6.0 (2026-03-21)
- 🔧 **干员面板**：API 请求参数修正为严格的 CamelCase（`instId`），解决面板同步时遇到的 400 缺少实例 ID 报错。
- 🔧 **干员面板**：为战斗面板 `/api/panel/char` 请求注入必需的上下文参数 `roleId`、`serverId`，解决战斗属性（生命、攻击、防御等）显示为横杠（`-`）的问题。
- 🔧 **推送稳定性**：自动签到推送的 ID 原型自动修补调整，规范为 `GroupMessage` 等，修复推送失败的问题。
